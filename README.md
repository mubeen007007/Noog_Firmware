# Noog Firmware

Firmware project for the NOOG board based on the `STM32F407VGT6`.

## Project Overview

This repository contains the STM32CubeIDE project for the NOOG firmware, including:

- STM32Cube-generated startup and HAL configuration
- FreeRTOS integration
- FATFS middleware
- Board-specific peripheral pin mapping from the hardware schematic

## MCU

- `STM32F407VGT6`
- Cortex-M4
- External HSE clock
- HAL-based project structure

## Configured Peripherals

- `SPI1`
- `SPI2`
- `SPI3`
- `I2C1`
- `I2C2`
- `I2C3`
- `UART4`
- `USART2`
- `USART3`
- Board control/status GPIOs

## Project Structure

- [Core](E:/NOOG%20Firmware/Core) - application code, interrupts, startup integration
- [Drivers](E:/NOOG%20Firmware/Drivers) - CMSIS and STM32 HAL drivers
- [FATFS](E:/NOOG%20Firmware/FATFS) - generated FATFS glue code
- [Middlewares](E:/NOOG%20Firmware/Middlewares) - FreeRTOS and FatFs middleware
- [NOOG Firmware.ioc](E:/NOOG%20Firmware/NOOG%20Firmware.ioc) - STM32CubeMX configuration

## Build Notes

This project builds with:

- STM32CubeIDE generated makefiles
- `arm-none-eabi-gcc`

Local build output is placed under `Debug/` and is ignored by Git.
